## The Objective

Build a complete 5-pillar optimized affiliate marketing website for "boring but essential" household products (toilet plungers, shower curtains, extension cords, etc.). The site needed to demonstrate expertise through original testing data and be optimized for both traditional SEO and emerging AI discovery (GEO/LLMO).

**Key Requirements:**
- Static site generator (Astro 5 chosen)
- 17+ pages (3 complete reviews, 3 placeholder reviews, categories, support pages)
- 5-pillar visibility framework (SEO, SEvO, GEO, AGO, LLMO)
- Schema.org structured data throughout
- Sortable comparison tables
- Newsletter capture forms
- Working Amazon affiliate links

---

## The Struggle

### 1. Layout & Container Issues
The initial BaseLayout had a constrained container that made text look cramped on large screens. Hero text, newsletter CTAs, and footer content were all being squeezed into narrow columns (max-w-3xl).

**Solution:** Modified multiple pages to use wider containers (`max-w-4xl` and `w-full`) for text content while keeping visual hierarchy.

### 2. Component Styling Problems
- **PickCard badges** were positioned absolutely but breaking on smaller screens
- **CTA button text** wasn't showing white color due to Tailwind specificity issues
- **Newsletter forms** had inconsistent spacing

**Solution:** 
- Used `!text-white` with `!important` for button text
- Restructured badge positioning to use flex containers
- Standardized padding across newsletter variants (hero/inline/footer)

### 3. Amazon ASIN Verification
Discovered several Amazon ASINs in the original content were either:
- Pointing to outdated/discontinued products
- Using incorrect ASIN format
- Missing proper affiliate tags

**Solution:** Replaced all ASINs with current working products:
- Korky Plunger: B0FG47SHKB
- Neiko Plunger: B08V8LNR1W
- Simplehuman: B00LJ4M572
- And verified all used `boringbest-20` tag

### 4. Image Placeholder System
Needed a quick way to create placeholder images without external dependencies.

**Solution:** Created SVG-based placeholders saved as .jpg files with product names embedded:
- `/public/images/korky-plunger.jpg`
- `/public/images/neiko-plunger.jpg`
- `/public/images/simplehuman-plunger.jpg`

## The Breakthrough

### 5-Pillar Content Architecture
The biggest win was structuring content that serves multiple discovery channels simultaneously:

**Traditional SEO wins:**
- Semantic HTML with proper H1/H2/H3 hierarchy
- Meta descriptions with clear value propositions
- Sitemap.xml and robots.txt auto-generated
- BreadcrumbList schema for every page

**GEO/LLMO wins:**
- Original testing statistics ("47 hours of testing, 12 products evaluated")
- FAQ sections with direct, quotable answers
- "Last updated" timestamps on every review
- Clear methodology page demonstrating expertise

**SEvO wins:**
- Reddit-ready stats (can copy/paste into r/BuyItForLife)
- Shareable "testing credentials" displayed prominently
- Numbers-focused content (price, rating, testing hours)

### Sortable Table Component
Built a client-side sortable comparison table that:
- Works without JavaScript (progressive enhancement)
- Sorts by price, rating, or custom columns
- Includes accessible ARIA labels
- Responsive on mobile devices

## Key Decisions Made
